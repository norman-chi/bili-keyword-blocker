# B站关键词批量拉黑

一个油猴（Tampermonkey）脚本，通过关键词搜索 B 站视频或用户，批量将搜索结果中的博主加入黑名单。

# 需求背景

最近我在浏览B站时，总是会推送一堆擦边视频给我。即便我不停地拉黑作者，在我的推荐流里依然会出现很多新的视频，这让我非常困扰。
迫于无奈我用AI开发了这样一个脚本，可以帮助我批量拉黑用户。

## 功能

- 支持多关键词（Tag 形式管理）
- 支持两种搜索模式：视频搜索（拉黑视频作者）、用户搜索（直接拉黑搜索到的账号）
- 拉黑前预览博主列表，确认后再执行
- 视频搜索预览会展示命中的视频封面，方便判断博主是否需要拉黑
- 可在预览列表中移除不需要拉黑的博主
- 实时显示执行日志（成功 / 跳过 / 失败）
- 自动跳过已在黑名单的用户
- 请求间隔 800ms，降低风控风险

## 安装

1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)
2. 点击下方链接一键安装脚本：

   **[👉 点击安装](https://github.com/norman-chi/bili-keyword-blocker/raw/main/bili-keyword-blocker.user.js)**

3. 在弹出的 Tampermonkey 页面点击「安装」即可

## 使用方法

1. 登录 B 站，打开任意 B 站页面（如 `bilibili.com`）
2. 页面右上角会出现「🚫 关键词批量拉黑」面板
3. 在输入框输入关键词，按 **回车** 添加（可添加多个）
4. 选择搜索类型和抓取页数
5. 点击「① 获取博主列表」，等待搜索完成
6. 确认预览列表中的博主无误后，可先移除不需要拉黑的博主，再点击「② 确认并批量拉黑」

> 💡 建议先用 1 页预览，确认结果符合预期后再大量拉黑

## 注意事项

- 需要登录 B 站才能使用（脚本依赖登录 Cookie 中的 `bili_jct` 进行鉴权）
- 拉黑操作**不可批量撤销**，请谨慎确认博主列表
- 单次建议不超过 200 个，避免触发风控
- 本脚本仅供个人使用，请勿滥用

## 开发

脚本入口：`bili-keyword-blocker.user.js`

使用原生 JS + GM API，无需构建工具，直接编辑即可。
